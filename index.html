Artisanal, tailor made, hand crafted & custom made..

For the International Children's Games 2015 held in The Netherlands, we have build a custom application to register all 3000 event visitors, participants and VIP's. The application is used by an audience from all over the world. More then 75 cities will be participating in the Games.

The output from the App is used by the organisation for a wide range of purposes. For example for planning logistics, catering, maps, competition schedules, security plans and additional events.

Features

Athlete Matrix

With the Athlete Matrix each Head of Delegation can determine the status of it's registered athlete's. It calculates the current team status depending on the given team members and sports limits. Team and sport limits can individual be managed by the ICG Managers.

Multilevel Authentication

Multiple roles can be defined by the ICG Managers, they are also able to attached multiple features to each role. We call it: 'the multilevel authentication' feature.

Packages

The ICG Managers are having full control over every package that can be ordered.

Waiver Form Management

We created an uploading module with the ability for the ICG Managers to give instant feedback to the uploader of the waiver form. Uploading forms till 20MB each, requires a diskspace of more than 75Gb.

WYSIWYG

The ICG Managers are able to edit a lot of meta data. From simple text lines to help pages.

Logging

Each mail, each page request, database request and credit card transaction is fully logged.

Dashboards

Using modern front end web development technologies we are able to deliver a web application that works on every device.

In App Help

Click on the help buttons and you will get instant feedback on the page you are in.

Payments

You can purchase you're selected packages directly within the ICG App with a creditcard.

Security

We're always using the latest software and our workflow adapts to this. The App is handling blackhole callbacks, CSRF protection , form tampering protection, works on SSL only and more...

Performance

We're using modern techniques like Grunt on the front-end development part and we are also working with PHP Accelerators, Nginx and using Memcached. We currently have an 60 milliseconds average on page load time.

Workflow

We use Git for our software version control, but also to deploy our software branches at once on the multiple server platform we have. In our development workflow we use the DTAP way. Each new feature first has to be fully tested and accepted by the ICG Managers before we can deploy this new feature to the production platform.

Hosted in the Cloud
